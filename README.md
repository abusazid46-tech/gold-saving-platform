# Gold Saving Platform

A complete fintech platform for investing in digital gold, similar to Jar App.

## Features
- User registration with OTP
- KYC verification (PAN & Aadhaar)
- Buy/Sell digital gold with real-time prices
- Auto-save features (round-up, daily savings)
- Wallet system with Razorpay integration
- Admin dashboard
- React Native mobile app

## Tech Stack
- **Backend**: FastAPI, PostgreSQL, Redis
- **Mobile**: React Native
- **Admin**: React.js
- **Payments**: Razorpay
- **Deployment**: Docker, AWS
